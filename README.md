# Applications of Bioinformatics

This repository contains a collection of bioinformatics analyses performed using publicly available databases, web servers, and computational tools. The projects focus on understanding biological systems, disease mechanisms, therapeutic targets, and genomic data interpretation.

Topics Covered:
• Pathway Enrichment Analysis
• B-cell Epitope Prediction (IEDB)
• Glycoinformatics Analysis
• Gene Interaction Network Analysis (Cytoscape)
• Pharmacogenomics (PharmGKB)
• CRISPR Efficiency Prediction
• Neutrophil Phenotype Data Exploration
• Brain-specific Data Analysis
• Toxicogenomic Profiling

The repository serves as a learning portfolio demonstrating practical applications of bioinformatics in genomics, systems biology, immunology, and precision medicine.
